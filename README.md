# Airside Bus Logger PWA

Ready to upload to GitHub Pages.

## Included
- Login with manual Bus ID
- Starting and end-of-day kilometres
- Passenger clicker
- Arrivals and Departures
- Renamable timestamp buttons
- Flight number, airline and route
- Highlift and Crew tracking
- Flight log with Edit and Delete
- Live dashboard
- Shift history and CSV export
- Installable PWA with offline support

## Default password
`1234`

## Publish on GitHub Pages
1. Create a repository.
2. Upload all files and the `icons` folder.
3. Open **Settings > Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/root`, then save.

## Install on Android
Open the published page in Chrome and choose **Install app** or **Add to Home screen**.

Data is stored on the current device. Export CSV backups regularly.
